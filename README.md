<h1>🍔 <b>TechBurger Ordering App - Kettutytöt</b></h1>

This app is a learning environment for IT-engineer students. It depicts a burger joint website with ordering feature (no checkout).

<h2>🛠️ <b>Tech Stack Included</b></h2>

- React 19 (via Vite)
- TypeScript (for type-safe development)
- Tailwind CSS 4 (for rapid UI styling)
- Zustand (for state management)
- Vitest (for testing)


<h2>🚀 <b>Want to use this code locally? (Whatever rocks your boat man)</b></h2>

1. Clone to Local: Open your terminal and clone the repo:

    git clone <repo-url>

2. Install Dependencies:

    cd <project-folder>
    npm install

3. Run Development Server:

    npm run dev

<h2>📂<b> Project Structure</b></h2>

- src/components/: Components of the site are here.
- src/types.ts: This includes interfaces.
- src/App.tsx: The main entry point of your application.
- src/services/: Has the services used, like API
- src/views/: Here you find the 'sites' active on the website.
- public/: Static data.
